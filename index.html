❓ Help & Guide

🚀 Getting Started

MyWorkLog is a PWA for tracking your daily attendance and tasks. All data is stored locally on your device and optionally synced to a Google Sheet.

📋 Report Types
Entry 🟢 — Log your arrival time. Used to calculate daily work duration.
Exit 🔴 — Log your departure time. Work duration = Exit − Entry.
Task 📋 — Log time spent on a project. Project is required; description is optional.
⏱ Task Duration

Use the clock spinner to set hours and minutes, or tap a preset button (15 min, 30 min, 1h…). Alternatively switch to Time Range mode to enter start and end times.

📊 Summary Tab
📅
Day view — Shows entry/exit counts, task hours, and calculated work duration for that day.
📆
Month view — Shows totals across the month: total work hours and number of reported days.

Navigate using the ‹ › arrows to move between days or months.

🗑 Deleting Reports

Open the History tab to see the last 30 days of reports. Tap the 🗑 button on any report to delete it locally and from the connected Google Sheet.

⚙️ Setup — Google Sheets Sync
  1. Open your Google Sheet → Extensions → Apps Script
  2. Paste the provided google-apps-script.gs code and save
  3. Click Deploy → New deployment → Web App
  4. Set Execute as: Me and Who has access: Anyone
  5. Copy the Web App URL
  6. Open Settings ⚙️ in MyWorkLog and paste the URL
  7. Paste your Sheet URL to enable the direct link button

Three sheets are created automatically: WorkLog (all reports), Attendance (daily entry/exit/duration), Tasks (task log).

📴 Offline Mode

Reports submitted while offline are saved locally and queued for sync. When connectivity is restored, tap 🔄 Sync on the Report tab, or it syncs automatically.

📱 Install as App

MyWorkLog is a Progressive Web App (PWA). When prompted, tap Install to add it to your home screen for a native app experience — no app store required.

💡 Tips
👉
Swipe left/right on the main screen to switch between tabs.
👉
Tap 🕐 Now to instantly fill in the current date and time.
👉
New projects typed in the Task form are saved automatically to your project list.
👉
Use the version badge (top right) to reset the app if needed.
