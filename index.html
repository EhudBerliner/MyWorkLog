איך משתמשים

לוגיקות מרכזיות
⏱ חישוב משך יום
כניסה ויציאה לאותו תאריך נכתבות בגיליון Attendance באותה שורה. משך היום מחושב אוטומטית: יציאה פחות כניסה (פורמט HH:MM).
📴 מצב לא מקוון
דיווחים שנכשלו נשמרים בתור מקומי. בחזרת חיבור — מסונכרנים אוטומטית. לחץ סנכרן לסנכרון ידני.
🗑 מחיקה
מחיקה בהיסטוריה מוחקת מהאפליקציה ומהגיליון. אם אין חיבור — המחיקה בגיליון תבוצע בחיבור הבא.
🔄 איפוס אפליקציה
לחיצה על מספר הגרסה פותחת תפריט איפוס. ניתן לבחור מה לשמור: היסטוריה, הגדרות (Endpoint, גיליון, פרויקטים) ותור ממתין.
📋 פרויקטים
פרויקטים נשמרים אוטומטית בהזנתם. לניהול הרשימה: הגדרות → פרויקטים שמורים.
💾 שמירה מקומית
כל הדיווחים נשמרים תמיד במכשיר (localStorage). הגיליון הוא סנכרון נוסף — לא תאבד נתונים אם אין חיבור.
ניווט
☰ תפריט — ניווט בין לשוניות, מצב כהה/בהיר, הגדרות, עזרה
היסטוריה — 30 יום אחרונים, עד 200 דיווחים, עם מחיקה
סיכום — נתונים יומיים/חודשיים: כניסות, יציאות, משימות, שעות
החלקה — החלק ימינה/שמאלה בין לשוניות
🟢 כניסה
דיווח תחילת יום עבודה.
נרשם ב-WorkLog וב-Attendance בעמודת שעת הכניסה לאותו תאריך.
🔴 יציאה
דיווח סיום יום עבודה.
נרשם ב-Attendance בשורה של אותו תאריך ומחשב אוטומטית משך היום.
📋 משימה
דיווח עבודה על משימה ספציפית.
פרויקט — חובה. תיאור — אופציונלי.
משך זמן — שעות ודקות (00:15 עד 24:00).
טווח שעות — שעת התחלה ← שעת סיום.
נרשם ב-WorkLog וב-Tasks.
📅 דיווח רטרואקטיבי
שנה את התאריך והשעה לפני השליחה כדי לדווח על פעולה שהתרחשה בעבר.
שלבי הגדרת Google Apps Script
  1. פתח Google Sheets חדש
  2. לחץ Extensions → Apps Script
  3. מחק קוד קיים — הדבק קוד מהקובץ google-apps-script.gs
  4. שמור (Ctrl+S)
  5. Deploy → New deployment
  6. Type: Web App
  7. Execute as: Me
  8. Who has access: Anyone
  9. לחץ Deploy ← אשר הרשאות
  10. העתק את Web App URL
  11. הגדרות אפליקציה → הדבק URL → שמור
⚠️ לאחר שינוי קוד — יש ליצור New Deployment ולא לעדכן קיים. ה-URL ישתנה!
גיליונות שנוצרים אוטומטית
📋 WorkLog
כל הדיווחים — כניסה, יציאה ומשימות.
עמודות: Timestamp · Report_Date · Report_Time · Category · Description · Project · Record_ID
🕐 Attendance
כניסה + יציאה לכל יום — באותה שורה. משך היום מחושב אוטומטית.
עמודות: Report_Date · שעת כניסה · שעת יציאה · משך יום עבודה
✅ Tasks
דיווחי משימות בלבד.
עמודות: Report_Date · משך/טווח · Project · Description
זרימת נתונים
דיווח כניסה/יציאה → WorkLog + Attendance (אותה שורה לפי תאריך)
דיווח משימה → WorkLog + Tasks
מחיקה → מוחק מ-WorkLog ומהגיליון המתאים
